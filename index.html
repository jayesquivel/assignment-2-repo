How to Apply CSS to HTML.

Here's a quick tutourial on how to add an external CSS file in your HTML document. You can start by using the <link> element within the <head> section of your HTML document. The <link> element should also have the rel attribute set to "stylesheet" and the href attribute pointing to the path of your CSS file. Here's an example on how that would look: 

          <head>
              <link rel="stylesheet" href="styles.css">
          </head>

How to Use JavaScript in a Webpage

Here's another short tutourial on how to include an external JavaScript file into your HTML document. You can start by using the <script> tag with the src attribute. You're going to want to place this tag before the closing </body> tag or within the <head> section. You can also add JavaScript code directly within the HTML document by simply inclosing your JavaScript code with the <script> tag. Here's an example for both situations: 

            <!-- Linking to an external JavaScript file -->
            <head>
                <script src="app.js"></script>
            </head>
            
              <!-- Writing JavaScript within an HTML file -->
              <body>
                  <script>
                  console.log('Hello, world!');
                  </script>
              </body>
